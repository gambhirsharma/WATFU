## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

## How to use
- Put `CHATGPT API KEY` in `.env` file
```.env
OPENAI_API_KEY="...."
```

- make changes in `data/grocery.json` accordingly
